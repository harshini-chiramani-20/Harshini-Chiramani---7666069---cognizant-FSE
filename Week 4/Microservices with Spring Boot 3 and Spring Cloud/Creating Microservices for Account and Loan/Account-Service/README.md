# Account Service

## Objective

Create a Spring Boot microservice to provide account details.

## Technologies Used

- Java 17
- Spring Boot 4.1.0
- Spring Web
- Maven

## Endpoint

GET /account

## Sample Output

```json
{
  "accountNumber": 1001,
  "accountHolder": "Harshini",
  "balance": 50000.0
}
```

## Outcome

Successfully created a REST microservice that returns account details.
