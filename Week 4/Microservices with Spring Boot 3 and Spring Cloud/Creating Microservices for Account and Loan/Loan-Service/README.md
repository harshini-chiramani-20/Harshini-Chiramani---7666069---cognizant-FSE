# Loan Service

## Objective

Create a Spring Boot microservice to provide loan details.

## Technologies Used

- Java 17
- Spring Boot 4.1.0
- Spring Web
- Maven

## Endpoint

GET /loan

## Sample Output

```json
{
  "loanNumber": 2001,
  "borrowerName": "Harshini",
  "loanAmount": 250000.0
}
```

## Outcome

Successfully created a REST microservice that returns loan details.
