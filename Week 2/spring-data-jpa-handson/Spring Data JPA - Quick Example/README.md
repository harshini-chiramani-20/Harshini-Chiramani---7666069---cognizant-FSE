# Spring Data JPA - Quick Example

## Objective

To create a Spring Boot application integrated with Spring Data JPA and MySQL.

## Tools Used

- Java 17
- Spring Boot
- Spring Data JPA
- Maven
- MySQL

## Procedure

1. Created a Spring Boot project.
2. Added Spring Data JPA dependency.
3. Configured MySQL in application.properties.
4. Connected the application with MySQL.
5. Executed the Spring Boot application.

## Expected Output

Application started successfully.
