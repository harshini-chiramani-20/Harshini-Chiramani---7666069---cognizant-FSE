# Create Spring Web Project using Maven

## Objective

Create a basic Spring Boot Web project using Maven and verify that the application starts successfully.

## Tools Used

- Java 17
- Spring Boot
- Maven
- Spring Initializr
- VS Code

## Dependencies

- Spring Web
- Spring Boot DevTools

## Steps Performed

1. Created the project using Spring Initializr.
2. Imported the project into VS Code.
3. Built the project using Maven.
4. Started the Spring Boot application.
5. Verified the application is running on port 8080.

## Expected Output

Application starts successfully with embedded Tomcat.

## Outcome

Successfully created and executed a Spring Boot Web project using Maven.
