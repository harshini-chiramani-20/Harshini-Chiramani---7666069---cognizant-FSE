# REST - Country Web Service

## Objective

Create a REST endpoint that returns a Country object in JSON format.

## Technologies Used

- Java 17
- Spring Boot 3
- Spring Web
- Maven

## Endpoint

GET /country

## Response

{
"code": "IN",
"name": "India"
}

## Outcome

Successfully created a REST API that returns a Country object in JSON format using Spring Boot REST Controller.
