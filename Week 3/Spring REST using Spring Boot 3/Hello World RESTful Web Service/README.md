# Hello World RESTful Web Service

## Objective

Create a simple RESTful web service using Spring Boot that returns "Hello World!!".

## Technologies Used

- Java 17
- Spring Boot 4
- Spring Web
- Maven

## Project Structure

```
Code/
└── spring-rest-handson/

Output/
├── Terminal.png
└── Browser.png

README.md
```

## Steps Performed

1. Created a Spring Boot Maven project.
2. Added the Spring Web dependency.
3. Created a REST Controller.
4. Implemented a GET endpoint `/hello`.
5. Ran the Spring Boot application.
6. Verified the response in the browser.

## Endpoint

```
GET /hello
```

## Expected Output

```
Hello World!!
```

## Outcome

Successfully created and tested a basic RESTful web service using Spring Boot.
