# Exercise 7: Financial Forecasting

## Objective

Implement a financial forecasting model using recursion to predict future revenue based on a fixed annual growth rate.

## Files

- FinancialForecast.java

## Algorithm Used

- Recursion

## Compilation

```bash
javac FinancialForecast.java
```

## Execution

```bash
java FinancialForecast
```

## Sample Output

```
Current Revenue : 10000.0
Growth Rate : 10.0%
Forecast after 5 years : 16105.10
```

## Learning Outcome

- Understand recursion.
- Apply recursion to financial forecasting.
- Predict future values using growth rates.
