# Exercise: E-commerce Platform Search Function

## Objective

Implement search functionality for an e-commerce platform using Linear Search and Binary Search algorithms.

## Files

- Product.java
- SearchFunction.java

## Algorithms Used

1. Linear Search
2. Binary Search

## Compilation

```bash
javac *.java
```

## Execution

```bash
java SearchFunction
```

## Expected Output

```
Linear Search Result:
Tablet found in Electronics

Binary Search Result:
Tablet found in Electronics
```

## Learning Outcome

- Understand Linear Search.
- Understand Binary Search.
- Compare searching techniques.
- Improve search efficiency in large datasets.
