# Verifying Interactions using Mockito

## Objective

To verify that methods on mock objects are invoked correctly using Mockito's `verify()` method.

## Files

- ExternalApi.java
- NotificationService.java
- NotificationServiceTest.java

## Concepts Used

- Mockito
- verify()
- Mock Objects
- JUnit 5

## Command

```bash
mvn test
```

## Expected Output

```
Tests run: 1
Failures: 0
Errors: 0
BUILD SUCCESS
```

## Learning Outcome

- Create mock objects.
- Verify method invocations.
- Test interactions between objects using Mockito.
